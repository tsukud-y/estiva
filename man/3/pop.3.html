POP(3)                ESTIVA ライブラリ関数マニュアル                  POP(3)

名称
     POP - スタック(待ち行列)からエントリをポップする

ライブラリ
     ESTIVA ライブラリ (libestiva, -lestiva)

書式
     #include "estiva/que.h"

     void *
     pop(que q, object e);

解説
     pop(q,e)はスタック(待ち行列)qからエントリeをポップする。
     ここにおけるeは変数でさえあれば何でも構わない。

例1
#include "estiva/que.h"

     main(){
            static que *q;
            int c = 'C';
            initq(q);
            pushq(q,c);
            pop(q,c);           
     }

例2
#include "estiva/que.h"

     main(){
           static que *q;
           char *word = "hello";
           initq(q);
           push(q,word);
           pop(q,word);
     }

例3
#include "estiva/que.h"
 
typedef struct { char *key; double val;} element;

     main(){
           static que *q;
           element e;
           e.key = "e"; e.val = 2.718;
           push(q,e);
           pop(q,e);
     }

戻り値
     エントリeへのポインタです。スタック(待ち行列)が
     空の場合はNULLが返ります。

関連項目
