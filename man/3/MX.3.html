MX(3)              ESTIVA ライブラリ関数マニュアル                MX(3)

名称
     MX - 疎行列アクセスハンドル

ライブラリ
     ESTIVA ライブラリ (libestiva, -lestiva)

書式
     #include 

     MX *A;

解説
     MX ハンドルは疎行列型のハンドルである。
     疎行列ハンドル vと、行数 m, 列数 n を引数として渡す
     ことにより mx() マクロで疎行列の要素にアクセスする
     ことが可能である。
     
     MX ハンドルは initmx() マクロを用いて一度初期化する
     必要が有る。

     以下の例のように static 宣言が必要である。
     これは関数を一度抜けても、値が保存されていることが、
     initmx() で利用されているからである。

例
#include 

     main(){
            static MX *A;
            initmx(A,1000+1,8);
	    mx(A,1,1) = 0.1;
     }

戻り値
     MX 型は 構造体の typedef なので、戻り値はない。

関連項目
     mx(3), initmx(3)
