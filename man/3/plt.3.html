PLT(3)                  ESTIVA ライブラリ関数マニュアル                  PLT(3)

名称
     plt - データのプロット

ライブラリ
     ESTIVA ライブラリ (libestiva, -lestiva)

書式
     #include 

     void
     plt(FILE *fp, xyc *Mid, xyc *Z, nde *N, double *p);

解説
     plt() 関数は gnuplot 形式のデータ出力をファイルポインタ
     fp に対して行います。
     ここに
     Mid ... 中点節点対応表
     Z   ... 節点-座標対応表
     N   ... 要素-節点対応表
     p   ... プロットするデータ

     ポインタ p は ary1() で確保された一次元配列である必要が
     あります。
     pの次元から、それが p0(0次補間)データであるか、
     p1(1次補間)データであるか、流速データであるかを
     plt() 関数は判断します。

     p0データは p1補間をして、出力します。
     また、 -conta [n] コマンドラインオプションを用いることに
     より、等値線の出力を得ることもできます。
     ここに、省略可能の引数 [n] は等値線の本数です。
     等値線の本数はデフォルトでは 20 本です。

     fp == NULL の場合 gnuplot を呼びだし、画面に表示を行います。

     中点節点を用いたデータでない場合は Mid == NULL でも構いません。

戻り値
     戻り値はありません。

関連項目
     ary1(3), xyc(3), nde(3)
