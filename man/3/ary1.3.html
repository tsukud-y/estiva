ARY1(3)              ESTIVA ライブラリ関数マニュアル                ARY1(3)

名称
     ary1 - 1次元配列を動的に確保するマクロ

ライブラリ
     ESTIVA ライブラリ (libestiva, -lestiva)

書式
     #include "estiva/ary.h"

     void
     ary1(void *v, long n);

解説
     ary1()マクロは1次元配列を動的に確保する。
     基本的に malloc() しているのと大差はないが、配列の内容が
     ゼロクリアされるのと、 dim1()関数によって、確保した配列の
     要素の数が得られる。
     また、dim0()関数によって、配列要素の1元のサイズが得られる。

     以下の例のように static 宣言が必要である。
     これは関数を一度抜けても、値が保存されていることが、
     ary1() で利用されているからである。


例
#include "estiva/ary.h"

     main(){
            static double *u;
            ary1(u,100+1);
     }

戻り値
     戻り値はありません。

関連項目
     ary2(3), dim0(3), dim1(3)
