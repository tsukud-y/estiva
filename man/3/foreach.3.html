FOREACH(3)           ESTIVA ライブラリ関数マニュアル                FOREACH(3)

名称
    foreach()  - オブジェクト汎用形 forマクロ
    end        - foreach() マクロの終端子

ライブラリ
     ESTIVA ライブラリ (libestiva, -lestiva)

書式	
     #include "estiva/foreach.h"

     foreach(obj),&p1,&p2,&p3,...end { ;...; }

解説
     foreach()マクロは csh(1) の foreach 制御文に似た
     動作をします。ここにポインタ &p1, &p2,&p3 は obj 型の
     ポインタである必要があります。

     end 識別子は名前のバッティングが起こる可能性が大いにありますが、
     その場合は、 foreach.h インクルードファイルの内容を
     参照して、適当に #undef, #define すると良いでしょう。

戻り値
     戻り値はありません。

関連項目
     foreach(3), end(3)
