FOREACH(3)           ESTIVA ライブラリ関数マニュアル                FOREACH(3)

名称
    foreach()  - オブジェクト汎用形 forマクロ

ライブラリ
     ESTIVA ライブラリ (libestiva, -lestiva)

書式	
     #include "estiva/foreach.h"

     foreach(obj,&p1,&p2,&p3) { ;;;; }

解説
     foreach()マクロは csh(1) の foreach 制御文に似た
     動作をします。ここにポインタ &p1, &p2,&p3 は obj 型の
     ポインタである必要があります。
     &p4,&p5,&p5と、引数はいくらでも増やせます。

戻り値
     戻り値はありません。

関連項目
     foreach(3)
