GETOP(3 )              ESTIVA ライブラリ関数マニュアル                 GETOP(3)

名称
     stdfp - 入力用ファイルストリーム

ライブラリ
     ESTIVA ライブラリ (libestiva, -lestiva)

書式
     #include 

     FILE *
     stdfp(void);

解説
     stdfp() は入力用ファイルストリームを適当に選択してファイル
     ポインタを返します。

     1. -f オプションで指定されたファイルを開くことを試みます。
     2. -f オプションが指定されていない場合は、コマンドラインオプション
        を全てスキャンし、開けるファイルをオープンします。
     3. 1,2,共に適当なファイルが該当しない場合は、標準入力を開きます。

     stdfp()関数を呼び出す前に一度initop(3)を呼び出す必要があります。
     もしinitop()ルーチンを呼び出すことなくstdfp()を呼び出した場合には、
     コマンドライン引数がなにもないままmain()関数を呼び出したと同じ
     動作をstdfp()は行います。具体的にはstdfp()はこの場合常に標準入力を
     返します。

戻り値
     戻り値は標準入力に準ずるファイルポインタです。

関連項目
     initop(3)
