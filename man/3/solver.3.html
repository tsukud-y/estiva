SOLVER(3)              ESTIVA ライブラリ関数マニュアル              SOLVER(3)

名称
     solver - 連立一次方程式を解く

ライブラリ
     ESTIVA ライブラリ (libestiva, -lestiva)

書式
     #include "estiva/solver.h"

     void
     solver(MX *A, double *x, double *b);

解説
     solver() 関数は連立一次方程式 Ax = b を満たす x を求めます。
     引数 A は MX型のポインタです。

     コマンドラインオプション -solver に適当なアルゴリズム名を
     指定することが出来ます。initop()関数による引数データの
     イニシャライズが必要です。
     
     -solver pcgs       ... 前処理付き双共役勾配法(デフォルト)
     -solver bicg       ... BiCG法
     -solver gauss      ... ガウスの消去法
     -solver blu        ... 帯LU分解
     -solver cgs        ... CGS法 
     -solver bicgstab   ... BiCGstab法
     -solver qmr        ... QMR法
     指定なし (qmr)

戻り値
     戻り値はありません。

関連項目
     ary1(3), initmx(3), initop(3)
