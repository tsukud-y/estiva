INITMX(3)              ESTIVA ライブラリ関数マニュアル                INITMX(3)

名称
     initmx - 疎行列初期化マクロ

ライブラリ
     ESTIVA ライブラリ (libestiva, -lestiva)

書式
     #include "estiva/mx.h"

     void
     initmx(void *v, long n, long k);

解説
     initmx()マクロは疎行列を動的に確保する。
     疎行列ハンドル vと、次元 n および 一行あたりの
     非ゼロ成分の個数 k を引数として渡す必要がある。

     以下の例のように static 宣言が必要である。
     これは関数を一度抜けても、値が保存されていることが、
     initmx() で利用されているからである。


例
#include "estiva/mx.h"

     main(){
            static MX *A;
            initmx(A,1000+1,8);
     }

戻り値
     戻り値はありません。

関連項目
     solver(3), mx(3)
