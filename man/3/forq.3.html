FORQ(3)               ESTIVA ライブラリ関数マニュアル                 FORQ(3)

名称
     FORQ - 待ち行列の内容でループ

ライブラリ
     ESTIVA ライブラリ (libestiva, -lestiva)

書式
     #include "estiva/que.h"

     forq(que q, void *f)

解説
     forq(q,f)は待ち行列qからポインタfを一つずつ取り出して
     ループを行う。

例
#include "estiva/que.h"
 
typedef struct { char *key; double val;} element;

     main(){
           static que *q;
           element e, *f;
           e.key = "e"; e.val = 2.718;
           push(q,e);
           e.key = "pi"; e.val = 3.14;
           push(q,e);

           forq(q,f) printf("%s %f\n", f->key, f->val);              
     }

戻り値
     ループ文です。戻り値はありません。
