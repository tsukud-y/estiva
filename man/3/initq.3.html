INITQ(3)              ESTIVA ライブラリ関数マニュアル                INITQ(3)

名称
     initq - 待ち行列初期化マクロ

ライブラリ
     ESTIVA ライブラリ (libestiva, -lestiva)

書式
     #include "estiva/que.h"

     void
     initq(que q);

解説
     initq()マクロは待ち行列を動的に確保する。
     待ち行列ハンドル　q　を引数として渡す必要がある。

     以下の例のように static 宣言が必要である。
     これは関数を一度抜けても、値が保存されていることが、
     initq() で利用されているからである。


例
#include "estiva/que.h"

     main(){
            static que *q;
            initq(q);
     }

戻り値
     戻り値はありません。

関連項目
