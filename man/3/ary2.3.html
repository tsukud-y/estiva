ARY2(3)              ESTIVA ライブラリ関数マニュアル                ARY2(3)

名称
     ary2 - 2次元配列を動的に確保するマクロ

ライブラリ
     ESTIVA ライブラリ (libestiva, -lestiva)

書式
     #include 

     void
     ary2(void *v, long m, long n);

解説
     ary2()マクロは2次元配列を動的に確保する。
     周知のように、C言語には動的に2次元配列を確保する方法はない。
     しかし、イリィッフィイベクタとして、2次元配列と同じアクセス方法を
     実現することが出来る。
     これは A[1][2] = 1.0; 等として、配列と同じ表現でアクセス出来る
     ということを意味している。
     
     1度目に呼び出された時、または配列のサイズが変わった場合に限り、
     配列の内容はゼロクリアされる。

     dim1(), dim2()関数によって、確保した配列の元の数が得られる。
     また、dim0()関数によって、配列要素の1元のサイズが得られる。

     以下の例の通り、 ary1()と同様に、static 修飾が必要である。
     これは関数を一度抜けても、値が保存されていることが、
     ary2() で利用されているからである。


例
#include 

     main(){
            static double **u;
            ary2(A,100+1,100+1);
     }

戻り値
     戻り値はありません。

関連項目
     ary1(3), dim0(3), dim1(3), dim2(3)
