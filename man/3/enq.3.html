ENQ(3)                ESTIVA ライブラリ関数マニュアル                  ENQ(3)

名称
     enq - 待ち行列にエントリを追加する

ライブラリ
     ESTIVA ライブラリ (libestiva, -lestiva)

書式
     #include "estiva/que.h"

     void
     enq(que q, object e);

解説
     enq(q,e)は待ち行列qにエントリeを追加する。
     ここにおけるeは変数でさえあれば何でも構わない。

例1
#include "estiva/que.h"

     main(){
            static que *q;
            int c = 'C';
            initq(q);
            enq(q,c);
     }

例2
#include "estiva/que.h"

     main(){
           static que *q;
           char *word = "hello";
           initq(q);
           enq(q,word);
     }

例3
#include "estiva/que.h"
 
typedef struct { char *key; double val;} element;

     main(){
           static que *q;
           element e;
           e.key = "e"; e.val = 2.718;
           enq(q,e);
     }

戻り値
     戻り値はありません。
