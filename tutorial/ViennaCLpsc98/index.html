ViennaCLでPSC98に挑戦

まずはViennaCLのインストール:

ダウンロード

展開 

$ gzip -cd ViennaCL-1.1.2.tar.gz | tar xvpf -

インストール 

$ cd ViennaCL-1.1.2
# mv viennacl /usr/include
以下の2つのファイルをダウンロード

main.cpp

yosen.c

yosen.cは、適当に加工する必要がある。 というのは、htmlのタグを取り除いて、純粋なCのソースファイルにする必要がある。

コンパイル 
$ cc -c yosen.c
$ c++ main.cpp yosen.o -lOpenCL -lm
テストラン 
$ ./a.out
Problem NO : 0
|b - Ax|_inf = 3.97654e-07  (OK)
