これは何?

estiva は科学技術計算に用いる事を想定した 行列演算ライブラリです。
比較的規模の大きな問題でも計算を遂行可能な 高性能なものとなっています。
プログラマの立場に立ち戻って、簡易なインターフェースと なっています。

名前について

essential value を短縮した造語ですが、
ラテン系の言葉で"夏の間の"という単語と同じ綴りにしました。
偶然ですが、festival という単語の頭の f と 末尾の l を取り除いた綴りです。

バージョンについて

2009年9月末リリースのバージョンは 0.9 です。
リリースバージョンは 0.9, 0.99, 0.999 と 続く予定です。
開発者の私が死んだら バージョン 1.0 となり、 以降のバージョンについては関知しません。

API のリスト 

initop()     - コマンドラインオプションAPI のイニシャライズ関数
defop()      - 指定したコマンドラインオプションが定義されているかどうかを判定する
getop()      - 指定したコマンドラインオプションに対応する引数を取得する
stdfp()      - 入力用ファイルストリーム
ofp()        - 出力用ファイルストリーム
ary1()       - 1次元配列を動的に確保するマクロ
ary2()       - 2次元配列を動的に確保するマクロ
dim0()       - 1次元、2次元配列の要素のサイズを返す
dim1()       - 1次元、2次元配列の行数を返す
dim2()       - 2次元配列の列数を返す
MX型         - 疎行列ハンドル型
initmx()     - 疎行列初期化マクロ
mx()         - 疎行列アクセスマクロ
pltmx()      - 疎行列のプロット
clonemx()    - 疎行列の複製
clearmx()    - 疎行列のゼロクリア
solver()     - 連立1次方程式を解く
esolver()    - 固有値問題を解く
xyc型        - 節点-座標対応表型
nde型        - 要素-節点対応表型
pushxyc()    - スタック(待ち行列)にxycエントリをプッシュする
genmesh()    - デローニー三角分割によるメッシュジェネレーション
fp2xyc()     - ストリームからxyc形式のデータを生成
xmesh()      - 有限要素格子データのビュー
fp2mesh()    - ストリームからmesh形式のデータを格納
delaunay()   - デローニー三角分割
rectmesh()   - 正方形領域の格子生成
fprintmesh() - ストリームへmesh形式のデータを出力
np1()        - 中点節点ジェネレータ
p2()         - 2次要素(中点節点)のジェネレーション
forgamma()   - 境界節点用 forマクロ
plt()        - データのプロット
tmpopen()    - 一時ファイルをオープンする
tmpname()    - 一時ファイルの名前を返す
tmpclose()   - 一時ファイルをクローズする
foreach()    - オブジェクト汎用形 forマクロ
end          - foreach() マクロの終端子
initq()      - 待ち行列初期化マクロ
enq()        - 待ち行列にエントリを追加する
push()       - スタック(待ち行列)にエントリをプッシュする
pop()        - スタック(待ち行列)からエントリをポップする
lup()        - 待ち行列の内容をルックアップする
forq()       - 待ち行列の内容でループ
fgetline()   - ファイルより一行読み込み
chomp()      - 文字列最後の改行文字を取り除く
fsize()      - ファイルのサイズの取得
flines()     - ファイルの行の数のカウント
forall()     - forマクロ
max()        - 最大値
min()        - 最小値
swap()       - 交換
cp()         - コピー

コマンドのリスト

xyc2msh      -  デローニー三角分割法を用いた有限要素メッシュジェネレーター
xmesh        -  有限要素メッシュを表示確認するコマンド
gnuplot2gif  -  gnuplot データを gif画像へ変換
